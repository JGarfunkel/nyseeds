Overview

The New York State Property Maintenance Code 2020 establishes standards for the maintenance and repair of existing buildings and properties to ensure safety, health, and overall quality.

The New York State Property Maintenance Code 2020 is based on the International Property Maintenance Code 2018 (IPMC 2018) with amendments and additions.
